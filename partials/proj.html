IoT Based Smart Grid

Sept 2015 - Nov 2015

Northeastern University

Developed an application using core java and Swing components, based on Singleton Ecosystem model.

Implemented Data Analytics dash board, at all hierarchy levels to analyze energy efficiency.

This project was aimed to develop a dynamic application, which would help in attaining efficient power management (Electricity) in smart cities. By implementing this model (Holonic Architecture) in Smart Cities, we can efficiently handle electricity power distribution and the loss of power, and minimize the dependency of Non-renewable resources and finally achieve “Go Green” power generation and distribution.
